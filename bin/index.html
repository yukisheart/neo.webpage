introducing the neoclock

always know what period it is,
never be late to class again.

welcome to the future of educational time!

the neoclock was originally thought of by my robotics teacher in order to supply an idea for my group's project.

well, the idea of a clock at least.

we created the neoclock using inspiration and a similar process to a teacher in a different state who employed Arduino and other tools to make her school's time system better.

you can find the link to her GitHub right here. use her github to find the svg files for the frame carvings.
